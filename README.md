# 🍔 Food Delivery UI

Projeto desenvolvido em **Flutter** como parte da prova de aproveitamento da disciplina **Desenvolvimento Mobile**.

O desafio consistia em reproduzir a interface de um aplicativo concorrente do iFood, com foco na fidelidade visual e na qualidade da implementação. O projeto **não realiza consumo de APIs**, sendo inteiramente voltado para o desenvolvimento da interface.

Os principais objetivos foram:

* 📱 Construção de layouts responsivos e fiéis ao design proposto;
* ⚡ Desenvolvimento com foco em performance e otimização da interface;
* 🧩 Aplicação de boas práticas de organização de código;
* 🎯 Utilização de componentes reutilizáveis para facilitar manutenção e escalabilidade.

Este projeto demonstra conhecimentos em composição de interfaces no Flutter, organização de widgets e desenvolvimento de aplicações com código limpo e de fácil manutenção.
